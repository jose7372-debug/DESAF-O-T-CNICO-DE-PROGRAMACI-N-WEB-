# Plataforma de Simuladores Matemáticos Interactivos

Este proyecto contiene dos simuladores interactivos basados en lógica matemática y física, desarrollados estrictamente con HTML5 semántico, CSS nativo (Box Model) y JavaScript Vanilla sin librerías externas.

## Características
1. **Simulador de Transferencia de Calor**: Implementa la Ley de Enfriamiento de Newton.
2. **Calculador de Combinaciones**: Resuelve combinaciones complejas de sorteos independientes con una función de factorial iterativa propia.

## Tecnologías Utilizadas
- HTML5 Semántico
- CSS3 (Grid System, Flexbox, Box Model Estándar)
- JavaScript (ES6+)

## Datos de Prueba del Enunciado
- **Ejercicio 1**: T0 = 120, Ts = 38, k = 0.45, t = 3. -> **Resultado esperado**: 59 °C.
- **Ejercicio 2**: Grupo 1 (59, 5), Grupo 2 (35, 1). -> **Resultado esperado**: 175,223,510 combinaciones.
